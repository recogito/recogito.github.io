Guides

The Guides section collects useful articles, tutorials and examples for Annotorious and RecogitoJS. If you want to learn about a specific topic that isn’t covered yet, please get in touch via the Gitter chat.

If you want to improve anything about a guide - or contribute a new one - please do so! Fork this site and send a pull request with your corrections or additions. The guides are located here in the repository.
