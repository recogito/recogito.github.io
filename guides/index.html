Guides

The Guides section collects useful articles, tutorials and examples for Annotorious and RecogitoJS. If you want to learn about a specific topic that isn’t covered yet, get in touch via our Gitter chat.

If you want to improve anything about a guide - or contribute a new one - please do so! Simply send us a pull request with your corrections or additions. The guides are part of the recogito/project-website-source GitHub repository and are located here.

Using Firebase for Storage

In order to store annotations, you need a server backend that is able to handle W3C Web Annotations. This guide provides a simple recipe for using Firebase, an online cloud storage service by Google, as your personal annotation server. Read more
